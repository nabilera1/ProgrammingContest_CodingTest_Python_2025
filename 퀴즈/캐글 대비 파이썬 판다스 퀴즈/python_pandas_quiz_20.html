Q1 · 리스트 내포 & 조건
Python
리스트 nums = [1, 2, 3, 4, 5, 6]에서
짝수만 골라 제곱한 리스트를 한 줄짜리 리스트 내포로 작성하세요.
힌트 [f(x) for x in ... if 조건] 형태를 떠올려 보세요.
정답 · 설명 보이기 ▶
정답 예시: [n**2 for n in nums if n % 2 == 0]
리스트 내포에서 for 뒤에 if 조건을 붙이면 해당 조건을 만족하는 원소만 선택할 수 있습니다.
여기서는 n % 2 == 0으로 짝수만 선택하고, 각 원소에 n**2를 적용합니다.
Q2 · enumerate 사용
Python
리스트 cols = ["age", "height", "weight"]가 있을 때,
0-age, 1-height, 2-weight 형태로 한 줄씩 출력하는 코드를 작성하세요.
힌트 for i, col in enumerate(cols): 패턴을 떠올려 보세요.
정답 · 설명 보이기 ▶
정답 예시: 
for i, col in enumerate(cols):
    print(f"{i}-{col}")
enumerate(iterable)는 (인덱스, 값) 튜플을 순회하게 해 줍니다.
기본 인덱스 시작값은 0이며, enumerate(cols, start=1)처럼 시작값을 바꿀 수도 있습니다.
Q3 · zip으로 두 리스트 합치기
Python
keys = ["math", "english", "science"],
values = [90, 85, 92]가 있을 때,
{"math": 90, "english": 85, "science": 92} 딕셔너리를 한 줄로 만드는 코드를 작성하세요.
힌트 dict(zip(keys, values)) 형태를 떠올려 보세요.
정답 · 설명 보이기 ▶
정답: scores = dict(zip(keys, values))
zip(keys, values)는 ("math", 90) 같은 튜플들을 생성하고,
dict()에 넣으면 이를 키-값 쌍으로 갖는 딕셔너리가 만들어집니다.
Q4 · sorted와 key 인자
Python
리스트 words = ["banana", "apple", "cherry", "kiwi"]를
문자열 길이 기준 오름차순으로 정렬한 새로운 리스트를 만드는 코드를 작성하세요.
힌트 sorted(words, key=len) 또는 lambda를 떠올려 보세요.
정답 · 설명 보이기 ▶
정답 예시: sorted_words = sorted(words, key=len)
sorted는 원본 리스트를 변경하지 않고 정렬된 새 리스트를 반환합니다.
key=len은 각 원소에 len()을 적용한 값을 기준으로 정렬하겠다는 의미입니다.
Q5 · 딕셔너리 내포
Python
nums = [1, 2, 3, 4]가 있을 때,
{1: 1, 2: 4, 3: 9, 4: 16}을 만드는 딕셔너리 내포를 작성하세요.
힌트 {k: v for ...} 형태를 떠올려 보세요.
정답 · 설명 보이기 ▶
정답: squares = {n: n**2 for n in nums}
딕셔너리 내포는 {키표현식: 값표현식 for ...} 패턴입니다.
각 n에 대해 키는 n, 값은 n**2를 주면 됩니다.
Q6 · 함수 정의와 기본값
Python
숫자 리스트와 스케일 계수를 받아,
각 원소에 계수를 곱한 새 리스트를 반환하는 함수 scale을 작성하세요.
계수의 기본값은 1.0으로 두세요.
힌트 def scale(nums, factor=1.0):를 떠올려 보세요.
정답 · 설명 보이기 ▶
정답 예시: 
def scale(nums, factor=1.0):
    return [x * factor for x in nums]
기본 인자값은 def 함수이름(매개변수=기본값) 형태로 지정합니다.
리스트 내포로 각 원소에 계수를 곱한 새 리스트를 간단하게 만들 수 있습니다.
Q7 · 리스트 슬라이싱
Python
data = [10, 20, 30, 40, 50, 60]가 있을 때,
가운데 [20, 30, 40, 50]만 잘라내는 슬라이싱을 작성하세요.
힌트 인덱스 1 이상, 5 미만 구간을 생각해 보세요.
정답 · 설명 보이기 ▶
정답: middle = data[1:5]
슬라이싱은 리스트[시작:끝]이며 끝 인덱스는 포함되지 않습니다.
인덱스 1,2,3,4에 해당하는 원소만 잘려 나옵니다.
Q8 · any / all
Python
리스트 scores = [70, 80, 95, 60]가 모두 50점 이상인지 확인하는 한 줄 코드를 작성하세요.
(결과는 True / False)
힌트 all(조건 for ...) 패턴을 떠올려 보세요.
정답 · 설명 보이기 ▶
정답 예시: ok = all(s >= 50 for s in scores)
all(iterable)은 모든 원소가 True면 True를 반환합니다.
s >= 50 for s in scores는 불리언 값들이 순서대로 만들어지는 제너레이터 표현식입니다.
Q9 · 파일 읽기 (with 문)
Python
"data.txt" 파일을 한 줄씩 읽어 화면에 출력하되,
파일을 자동으로 닫아 주는 코드 패턴(with 문)을 작성하세요.
힌트 with open(...) as f: 구조를 떠올려 보세요.
정답 · 설명 보이기 ▶
정답 예시: 
with open("data.txt", encoding="utf-8") as f:
    for line in f:
        print(line.strip())
with 블록이 끝나면 파일이 자동으로 닫히므로 안전합니다.
line.strip()으로 줄 끝의 줄바꿈 문자 등을 제거할 수 있습니다.
Q10 · 예외 처리
Python
사용자 입력을 int로 변환하려고 할 때,
변환에 실패하면 "잘못된 입력"이라고 출력하는 예외 처리 코드를 작성하세요.
힌트 try: / except ValueError: 구조를 떠올려 보세요.
정답 · 설명 보이기 ▶
정답 예시: 
s = input("숫자를 입력하세요: ")
try:
    n = int(s)
    print("입력한 숫자:", n)
except ValueError:
    print("잘못된 입력")
int() 변환이 실패하면 ValueError가 발생합니다.
try/except 블록으로 예외를 잡아 사용자에게 적절한 메시지를 보여 줄 수 있습니다.

f_adult = df[(df["age"] >= 18) & (df["sex"] == "female")]

Q17 · groupby와 집계
Pandas

어떤 코드를 써야 할까요?

Q20 · merge로 데이터 결합
Pandas
승객 기본 정보가 담긴 df_passengers와
각 승객의 티켓 가격이 담긴 df_fares가 있습니다.
두 데이터프레임 모두 "PassengerId" 열을 공통 키로 가지고 있을 때,
이를 기준으로 왼쪽 조인(left join)하는 코드를 작성하세요.
(왼쪽 기준: df_passengers)
힌트 pd.merge(..., how="left", on="PassengerId") 패턴을 떠올려 보세요.
정답 · 설명 보이기 ▶
정답 예시: 
df_merged = pd.merge(
    df_passengers,
    df_fares,
    how="left",
    on="PassengerId"
)
pd.merge(left, right, how="left", on="키열")은 SQL의 LEFT JOIN과 같습니다.
왼쪽 데이터프레임의 모든 행을 유지하면서, 키가 일치하는 오른쪽 정보만 붙입니다.
캐글에서 여러 CSV를 결합할 때 매우 자주 사용하는 패턴입니다.
