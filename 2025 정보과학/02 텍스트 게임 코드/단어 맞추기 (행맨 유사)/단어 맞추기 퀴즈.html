코드 이해도 점검 퀴즈 (5문제)

앞서 학습한 영단어 퀴즈 게임 코드의 작동 원리와 사용된 문자열 메서드에 대한 문제입니다.

문제 1. 문자열 합치기

코드에서 display_list = ['A', '_', 'P', 'L', '_'] 리스트를 "A _ P L _"처럼 글자 사이에 공백을 넣어 하나의 문자열로 합치려고 합니다. 빈칸에 들어갈 메서드는 무엇입니까?

result = " ".______(display_list)
문제 2. 입력 데이터 전처리

사용자가 실수로 공백과 함께 소문자 " t "를 입력했습니다. 이를 공백 없는 대문자 "T"로 한 번에 변환하기 위해 사용된 메서드 조합은 무엇입니까?

input_char.______().upper()
문제 3. 입력값 검증

사용자가 알파벳이 아닌 숫자나 특수문자(예: 1, !)를 입력했을 때, 이를 감지하고 차단하기 위해 사용된 조건 검사 메서드는 무엇입니까? (True/False 반환)

if not user_input.______(): print("알파벳만 입력하세요")
문제 4. 출력 꾸미기

print_header 함수에서 제목을 출력할 때, 40칸의 공간을 확보하고 글자를 가운데 정렬하기 위해 사용한 메서드는 무엇입니까?

title.______(40)
문제 5. 정답 확인 로직

화면에 표시된 문자열이 "G R A P E"(공백 포함)일 때, 이를 원래 정답 단어 "GRAPE"(공백 없음)와 비교하여 승리 여부를 판단하려고 합니다. 문자열 내의 모든 공백을 제거하기 위해 사용된 메서드와 사용법은 무엇입니까?

current_display.______(" ", "")
정답 및 해설
문제 1 join
설명: 리스트의 요소들을 지정된 구분자(공백)로 연결합니다.
문제 2 strip
설명: 문자열 양쪽 끝의 공백을 제거합니다.
문제 3 isalpha
설명: 문자열이 알파벳 문자로만 구성되어 있는지 확인합니다.
문제 4 center
설명: 지정된 너비만큼 공간을 확보하고 문자열을 가운데에 배치합니다.
문제 5 replace
설명: 특정 문자(공백)를 다른 문자(빈 문자열)로 대체하여 삭제합니다.
