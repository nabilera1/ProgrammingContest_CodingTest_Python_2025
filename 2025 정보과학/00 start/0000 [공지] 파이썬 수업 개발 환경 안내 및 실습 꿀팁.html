[공지] 파이썬 수업 개발 환경 안내 및 실습 꿀팁

안녕하세요, 여러분!
이번 학기 파이썬 프로그래밍 수업을 위한 개발 도구(IDE) 선택과 실습 시 주의사항을 안내해 드립니다.
미리 확인하여 원활한 수업 준비를 부탁드립니다.

1. 개발 환경(IDE) 선택 안내

파이썬 코드를 작성하고 실행하기 위해서는 편집기(IDE)가 필요합니다. 여러분은 다음 세 가지 중 본인에게 가장 편한 것 하나를 자유롭게 선택하여 사용하시면 됩니다.

  • PyCharm (파이참): 강력한 기능과 안정성을 갖춘 파이썬 전용 IDE
  • VS Code (비주얼 스튜디오 코드): 가볍고 확장성이 좋아 널리 쓰이는 에디터
  • Google Colab (구글 코랩): 별도 설치 없이 웹 브라우저에서 바로 실행 가능
📢 중요 알림
수업 시간의 모든 예제 설명과 화면 시연은 'PyCharm(파이참)'을 기준으로 진행됩니다.
다른 도구를 사용하셔도 무방하나, 메뉴 위치나 실행 단축키 등이 다를 수 있으니 이 점 참고해 주세요.
(아직 정하지 못했다면 수업과 동일하게 PyCharm 설치를 추천합니다!)

2. 파이썬 실습 시 자주 하는 실수 (미리 알면 좋은 꿀팁!)

코딩이 처음이라면 누구나 겪는 실수들입니다. 에러가 났을 때 당황하지 말고 아래 내용을 먼저 체크해 보세요.

  • 📏 들여쓰기(Indentation)는 생명입니다! 파이썬은 중괄호 {} 대신 들여쓰기로 코드 블록을 구분합니다. Tab 키나 Space 4칸을 일정하게 맞춰주세요.
  • 🔠 대소문자를 엄격하게 구분해요! Print와 print는 다릅니다. 변수명이나 함수명을 작성할 때 대소문자가 정확한지 확인하세요.
  • 🔍 콜론(:)을 잊지 마세요! if, for, while, def 등으로 시작하는 문장의 끝에는 반드시 콜론(:)이 붙어야 합니다.
  • 🔗 괄호의 짝을 맞춰주세요! 여는 괄호 ( [ { 가 있으면 반드시 닫는 괄호가 있어야 합니다.
준비 과정에서 설치가 안 되거나 궁금한 점이 있다면 언제든 질문해 주세요.
그럼 수업 시간에 만나요!
